Movie App

This movie app pulled frontend and backend together to create a functioning app that I created from the ground up. The API I created was used for the backend. I built the frontend using React. Users can create, edit, and delete their accounts. They can view a list of movies, as well as, more information about the movies like who the director is or what the genre is. Users can also add or remove movies to their favorites.

Technologies used: React, Heroku, and Netlify

Meet App

In this project I created an app that shows events and used Google Calendar API. Users will see events from different cities. They can choose how many they see on one page, filter cities, and see more information about each event. I also used data visualization so show information about the events in a pie chart and a scatter point graph. My development process during this project utilized unit testing, integration testing, and End-to-End testing. I found this process very useful.

Technologies used: Data visualization, Integration Testing, Unit Testing, and End-to-End Testing

Chat App

I created a simple chat app that has the basic features a user would expect from a chat app. Users can send and receive messages, send pictures from their library or take one on the spot, and send their location. Google firebase was used for the database, so that users can have an anonymous sign in, and their conversations saved and viewable offline.

Technologies used: Expo, Android Studio, React Native, Firebase
